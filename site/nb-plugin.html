This Netbeans plugin allows you to build your Codename One applications for Android locally.

This project is not affiliated with Codename One, the company.
It is a community project maintained by ArSi.
Builds produced by this project are not supported by Codename One.
Many thanks to Attila Kelemen for providing Gradle Support plugin.

Requirements

  • Android SDK

Features

  • Gradle build
  • On Save sync
  • Library sync
  • You can open the project direct in Android studio
  • CN1 native support
  • cn1libs support

How to

https://sourceforge.net/p/cn1offandrnb/wiki/Home.
